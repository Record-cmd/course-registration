# course-registration
-------------------------------------------------------------------------------------------
목적
-------------------------------------------------------------------------------------------
웹 개발에 필요한 기술스택의 활용법을 익히기 위해 진항한다. 

서버통신, 데이터베이스, API, 외부API연동까지 전반적인 웹개발 과정을 익힌다.

개발환경
-------------------------------------------------------------------------------------------
Vue, HTML, JavaScript, node.js, MongoDB, Visual Studio Code

내용
-------------------------------------------------------------------------------------------
사용자는 원하는 강의를 신청, 취소할 수 있으며 신청한 강의를 조회할 수 있다.

관리자는 강의를 등록, 수정, 삭제할 수 있다.
